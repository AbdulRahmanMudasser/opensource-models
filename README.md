# opensource-models

Model capability testing toolkit for evaluating reasoning capabilities.

## usage

run the main test script:

```bash
python open-source-models.py
```

format existing results:

```bash
python format-results.py
```

## structure

- `open-source-models.py` - main test runner
- `format-results.py` - result formatting utility
- `scripts/` - helper automation scripts
- `output/` - generated test results

## configuration

create `.env` and add your credentials.
